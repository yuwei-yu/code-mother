# 🤖 AI 零代码应用生成平台

## 📖 项目概述

本项目是一个基于 AI 的零代码应用生成平台，用户只需用自然语言描述需求，AI 即可自动生成完整的前端应用，并支持可视化编辑、一键部署与分享。平台采用 **Spring Boot 3 + LangChain4j + Vue 3** 技术栈，深度融合 **AI Agent** 与 **工作流引擎**，为开发者提供一套可复用的企业级 AI 应用开发范式。

项目不仅实现了从需求到代码的端到端生成，还涵盖了多租户 AI 服务架构、流式响应、微服务拆分、多级缓存、监控体系等后端核心设计，适合希望掌握 AI 工程化落地与高并发架构的技术人员深入研读。

## ⚡ 核心功能与技术实现

### 1. 🤖 智能代码生成
- **技术实现**：基于 LangChain4j 构建 AI Agent，通过**意图识别**与**工具调用**动态选择代码生成策略（如单页面生成、多文件项目生成）。
- **亮点**：采用 **流式输出** + **响应式编程**，实时推送 AI 的思考过程与生成结果，提升用户体验；通过 **函数调用** 机制让 AI 操作文件系统、模板引擎，生成可运行的项目结构。

### 2. 🎨 可视化编辑
- **技术实现**：生成的应用以 iframe 嵌入前端，前端通过 **postMessage** 与 AI 后台通信。用户选中页面元素后，可继续与 AI 对话修改样式或逻辑。
- **亮点**：实现了一套 **Vibe Coding** 式的交互闭环，AI 实时响应修改指令并更新页面，无需手动编写代码。

### 3. 🚀 一键部署与分享
- **技术实现**：集成容器化部署服务，通过 **Docker** 打包生成的应用，自动分配子域名并配置 Nginx 反向代理。部署完成后调用截图服务生成封面图。
- **亮点**：全流程自动化，支持源码下载，提供完整的 DevOps 示例（镜像构建、持续部署）。

### 4. 🏢 企业级管理后台
- **技术实现**：基于 Spring Boot 3 + Vue 3 实现 RBAC 权限管理、应用审核、精选应用设置。集成 **Prometheus + Grafana** 监控系统性能与 AI 调用指标。
- **亮点**：提供多维度业务监控（如生成成功率、平均耗时、成本统计），为运营调优提供数据支撑。

### 🔑 核心设计要点
- **AI 服务层**：基于 LangChain4j + LangGraph4j 构建可编排的 AI 工作流，支持复杂多轮对话与工具链调用。
- **响应式后端**：使用 Spring WebFlux 实现 AI 流式输出，结合 Project Reactor 提升吞吐量。
- **微服务架构**：采用 Spring Cloud Alibaba + Dubbo 拆分用户、应用、部署等模块，保证可扩展性。
- **多级缓存**：Redis + Caffeine 两级缓存降低 AI 服务调用成本，结合多租户隔离策略。
- **可观测性**：集成 Micrometer 追踪 AI 调用链路，实时采集 Token 消耗、响应时间等指标。

## ⚙️ 关键技术挑战与解决方案

| 挑战 | 解决方案 |
|------|----------|
| 🔴 如何让 AI 生成复杂项目结构？ | ✅ 设计工具调用机制，AI 通过创建文件、写入内容的原子操作组合生成多文件项目。 |
| 🔴 如何实现高性能流式响应？ | ✅ 采用 WebFlux + SSE，后端非阻塞推送，前端逐行渲染。 |
| 🔴 如何管理多租户的对话上下文？ | ✅ 基于 ThreadLocal 和 Redis 实现会话级记忆存储，支持长对话的 Token 截断与总结。 |
| 🔴 如何控制 AI 成本与性能？ | ✅ 引入多级缓存（相似请求命中）、模型降级（复杂任务走强模型）、请求聚合与限流。 |
| 🔴 如何保证部署服务的稳定性？ | ✅ 异步构建 + 容器隔离 + 健康检查，失败自动回滚并通知用户。 |
